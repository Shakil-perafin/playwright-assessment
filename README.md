#  Playwright Automation Assessment

This project is an automation testing framework developed using **Playwright** and **JavaScript** as part of the SQA Engineer assessment.

---

##  Project Overview

This framework automates the following test scenarios from [Automation Exercise](https://automationexercise.com):

### **Test Case 1 – Search Product**
1. Launch browser  
2. Navigate to [automationexercise.com](http://automationexercise.com)  
3. Verify home page is visible  
4. Click on “Products”  
5. Verify user is navigated to “All Products” page  
6. Search for a product  
7. Verify “Searched Products” section appears  
8. Validate that all relevant products are visible  

### **Test Case 2 – Add Product to Cart**
1. Launch browser  
2. Navigate to [automationexercise.com](http://automationexercise.com)  
3. Verify home page is visible  
4. Click “View Product” for any item  
5. Verify product details page is opened  
6. Increase quantity to 4  
7. Add to cart  
8. View cart and verify quantity is correct
